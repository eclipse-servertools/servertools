Adding the Apache Tomcat runtimes

Using the Preferences page, you can define the installed server runtime definitions for the Apache Tomcat server.Â To define these definitions, complete the following:

  1. From the Window menu, select Preferences.
  2. In the Preferences window, expand Server and then select Installed Runtimes.
  3. On the Installed Server Runtime Environments page, click Add. The New Runtime wizard opens.
  4. On the New Server Runtime page, expand Apache.
  5. Select one of the following supported Apache Tomcat servers:
    • Apache Tomcat v3.2
    • Apache Tomcat v4.0
    • Apache Tomcat v4.1
    • Apache Tomcat v5.0
  6. The Tomcat Server page opens when you click Next.
  7. In the Tomcat installation directory field, type or browse to the directory where you extracted the installation files of Apache Tomcat.
  8. Click Finish.
