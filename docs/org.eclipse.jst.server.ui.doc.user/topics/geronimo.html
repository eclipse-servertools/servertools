Creating an Apache Geronimo Server

The server tools feature supports the Apache Geronimo run-time environment, running locally. With Geronimo, you can test Web projects that contain servlets and JSPs or J2EE projects that also contain EJBs. To set up a server to test projects on Apache Geronimo, complete the following:

Prerequisite

  • Download Apache Geronimo from geronimo.apache.org
  • Extract the compressed file containing Apache Geronimo on your local file system.
  • Add the Apache Geronimo runtime.

Create Server

  1. In the Servers view, right-click and select New > Server to open the Define a New Server wizard.
  2. In the Select the server type list, select the Geronimo v1.0 Server.
    Tip: If you do not have a configured Apache Geronimo runtime, the Define a New Server wizard will prompt you for Geronimo runtime configurations.
  3. Click Next, and fill in your Address, Port, Administor Id and Adminstrator Password for your specific server configuration needs.
  4. Click Finish.
