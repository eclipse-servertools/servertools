Keyboard shortcuts for Server Tools

This documentation describes the keyboard shortcuts for tasks in the Servers view; and running and debugging artifacts on a server.

Table 1. Keyboard shortcuts for Server Tools
Function Key combination When
Debug on Server Alt + Shift + D, R In workbench
Run on Server Alt + Shift + X, R In workbench
Debug Ctrl + Alt + D In Servers view
Publish Ctrl + Alt + P In Servers view
Start Ctrl + Alt + R In Servers view
Stop Ctrl + Alt + S In Servers view
