Setting a default server

A default server is the server that you want to use when a project has more than one server defined.

To set a default server:

  1. Right-click on your project. The pop-up menu for this project appears.
  2. Select Properties > Server . The Server pane appears on the right.
  3. Under the Default server list, select the server that you want to become the default server.
  4. Click Apply and then OK to save your changes and close the Properties window.
