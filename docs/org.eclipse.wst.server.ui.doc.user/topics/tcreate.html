Creating, editing, and deleting servers

  • Creating a server
    You can create a server to identify the runtime environment that you want to use for testing your project resources.
  • Editing a server
    You can edit your server using a server editor. The fields that you change are dependant on what runtime environment the server is defined to use, or whether it is defined as a local or remote server.
  • Deleting a server
    You may delete a server from the Servers view.
