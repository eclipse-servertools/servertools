Defining the server preference

Using the Preferences page, you can define attribute used for testing projects by the servers.

To define this preference, complete the following steps:
  1. In the toolbar, select Window > Preferences.
  2. In the Preferences page, select Server.
  3. Select the Show Servers view when server state changes check box, if you want the Servers view to display on the workbench when there are any activities in the Servers view. For more details about the server states refer to the Servers View topic.
  4. Click Apply and then OK to save your changes.
