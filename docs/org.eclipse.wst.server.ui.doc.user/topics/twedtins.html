Editing a server

The server tools feature provides you with editors so that you may change your server configuration. To edit a server, you can do one of the following:

  • Double-click on the server in the Servers view.
  • Right-click on the server in the Servers view, and then select Open. Depending on what run-time environment the server is defined to use, or whether it is defined as a local or remote server, the fields that you can change may be different.
  • To save your changes, you may either select File > Save or press CTRL+S.
  • Make sure that the editor is closed before starting, restarting or adding and removing modules on the server
