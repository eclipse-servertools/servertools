Server ToolTip

Identifier:
org.eclipse.wst.server.ui.serverToolTip

Since:
3.0

Description:
This extension point provides a way to add additional text when the mouseover action is detected

Configuration Markup:

<!ELEMENT extension (serverToolTip)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>


<!ELEMENT serverToolTip EMPTY>

<!ATTLIST serverToolTip

id          CDATA #REQUIRED

class       CDATA #REQUIRED

serverTypes CDATA #REQUIRED>

  • id - specifies a unique identifier for this ToolTip provider
  • class - specifies the fully qualified name of the Java class that implements org.eclipse.wst.server.ui.internal.provisional.IServerToolTip
  • serverTypes - the server type ids that use this ServerToolTip

Examples:
The following is an example of the extension point: 

   <extension
         point="org.eclipse.wst.server.ui.serverToolTip">
      <serverToolTip
            id="com.example"
            serverTypes="*"
            class="com.example.ExampleToolTipProvider"/>
   </extension>

API Information:
This extension point is part of an interim API that is still under development and expected to change significantly before reaching stability. It is being made available at this early stage to solicit feedback from pioneering adopters on the understanding that any code that uses this API will almost certainly be broken (repeatedly) as the API evolves.
