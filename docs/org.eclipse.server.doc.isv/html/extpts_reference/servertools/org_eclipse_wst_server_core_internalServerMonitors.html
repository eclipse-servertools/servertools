Server Monitor

Identifier:
org.eclipse.wst.server.core.internalServerMonitor

Since:
1.0

Description:
This extension point is used to provide a new monitor for servers.

Configuration Markup:

<!ELEMENT extension (monitor+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>

  • point - a fully qualified identifier of the target extension point
  • id - an optional identifier of the extension instance
  • name - an optional name of the extension instance

<!ELEMENT monitor EMPTY>

<!ATTLIST monitor

id          CDATA #REQUIRED

name        CDATA #IMPLIED

description CDATA #IMPLIED

class       CDATA #REQUIRED>

  • id - specifies a unique identifier for this extension point
  • name - a translatable name used to identify this monitor
  • description - a translatable description of this monitor
  • class - specifies the fully qualified name of the Java class that implements org.eclipse.wst.server.core.internal.ServerMonitorDelegate This extension point is internal and should not be used by any other plugins.
