Server Profiler

Identifier:
org.eclipse.jst.server.core.serverProfilers

Since:
1.5

Description:
This extension point is used to provide support for profiling servers.

Configuration Markup:

<!ELEMENT extension (serverProfiler+)>

<!ATTLIST extension

point CDATA #REQUIRED

id    CDATA #IMPLIED

name  CDATA #IMPLIED>

  • point - a fully qualified identifier of the target extension point
  • id - an optional identifier of the extension instance
  • name - an optional name of the extension instance

<!ELEMENT serverProfiler EMPTY>

<!ATTLIST serverProfiler

id    CDATA #IMPLIED

name  CDATA #REQUIRED

class CDATA #REQUIRED>

  • id - specifies a unique identifier for this extension point
  • name - a translated name
  • class - a class that extends org.eclipse.jst.server.core.ServerProfilerDelegate

Examples:
The following is an example of a server profiler extension point: 

  <extension point="org.eclipse.jst.server.core.serverProfilers">
     <serverProfiler
        id="com.example.myProfiler"
        name="My Profiler"
        vmArgs="-XpiAgent"/>
  </extension>
