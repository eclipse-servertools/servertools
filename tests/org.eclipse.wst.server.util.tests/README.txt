About this plugin:
------------------
This plugin is a set of utilities used to create scenarios that would 
otherwise require code hack modifications to test a particular scenario.

It is not to be included as part of any product, and it shouldn't be 
included in any WTP BUILDS.

The use of this utilities can potentially damage important data in your 
workspace. Do not use this utilities unless you assume full responsability
of any damages(or improvements :)) to your workspace.
